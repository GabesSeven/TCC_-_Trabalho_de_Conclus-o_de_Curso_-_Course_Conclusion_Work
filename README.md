  <br>
  Capstone project was based on the construction of a Convolutional Neural Network (CNN) together with image processing techniques with the objective of detecting image tampering caused by the Seam Carving technique, taught by <a href="https://bv.fapesp.br/pt/pesquisador/666459/kelton-augusto-pontara-da-costa">PhD Kelton Augusto Pontara da Costa</a> from S√£o Paulo State University (Unesp). The project was developed in the following stages: <br>
  <ul>
    <li>choice of image database;</li>
    <li>choice of percentage rates for adding images from the database;</li>
    <li>development of image processing algorithms;</li>
    <li>development of the deep learning algorithm;</li>
    <li>application of techniques and measurement of accuracy;</li>
    <li>paper writing.</li>
  </ul>
